Personal Portfolio Website

A clean, animated, and scalable portfolio built with HTML, CSS & JavaScript

Project Overview

This project focuses on building a modern personal portfolio without frameworks, emphasizing performance, clarity, and maintainability.

Problem

Most portfolio templates are heavy, generic, and difficult to customize. I needed a lightweight solution that still felt premium and unique.

Solution

I designed a fully custom multi-page website using reusable components (navbar & footer), smooth transitions, and subtle animations.

Development Process

  • Planned information architecture and navigation
  • Created reusable UI components using JavaScript partial loading
  • Implemented scroll reveal, typing effect, and page transitions
  • Deployed using GitHub + Cloudflare Pages

Tech Stack

  • HTML5
  • CSS3 (Custom design system)
  • Vanilla JavaScript
  • Git & GitHub
  • Cloudflare Pages

Outcome

The result is a fast, professional, and scalable portfolio that clearly communicates skills and experience while remaining easy to maintain.
